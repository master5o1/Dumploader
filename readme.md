Dumploader
=======

Dumploader is a file-uploading and pasting service.
It is modelled after and loosely clones [Omploader](http://omploader.org/).

Requirements
------------
* Node.js (0.6.13)
* MongoDB (Shell version is 2.0.2)
* Express (2.5.8)
* Mongolian DeadBeef (0.1.17)
* ImageMagick (0.1.2)
* Jade (0.22.0)
